`reset` состоит из 2 частей.
## reset_base.scss

Предназначен для сброса общих стилей страницы.

## reset_form.scss

Предназначен для сброса стилей form, input, button, textarea, select
и всевдо элементов ::placeholder. Так же убирает outline.
